# Changelog Rules

Seluruh perubahan wajib dicatat.

* Nomor versi
* Tanggal
* Jenis perubahan
* Ringkasan perubahan

# Semantic Version Rules

MAJOR

Perubahan besar yang tidak kompatibel.

MINOR

Penambahan fitur baru.

PATCH

Perbaikan bug.
