# Roadmap

Pengembangan menggunakan **Sprint Based Development**. Satu sprint menghasilkan fitur yang siap digunakan. Tidak melanjutkan sprint berikutnya sebelum sprint saat ini selesai dan tervalidasi.

| Sprint | Fokus | Status |
|--------|-------|--------|
| **Sprint 0** | Project Setup (Next.js, Tailwind v4, Prisma, Supabase, shadcn/ui) | ✅ Selesai |
| **Sprint 1** | Authentication (Login, Register, Logout, Middleware, Session) | ✅ Selesai |
| **Sprint 2** | Manajemen Keluarga (Family CRUD, Anggota, Undangan) | ✅ Selesai |
| **Sprint 3** | **Master Data (Kategori, Dompet, Pengaturan)** | 🚧 Sedang dikerjakan |
| **Sprint 4** | Layout Dashboard (Sidebar, Navbar, Theme Switch) | ⏳ |
| **Sprint 5** | Dashboard KPI + Grafik | ⏳ |
| **Sprint 6** | Manajemen Pemasukan (CRUD, Upload, Filter, Export) | ⏳ |
| **Sprint 7** | Manajemen Pengeluaran (CRUD, Upload, Filter, Export) | ⏳ |
| **Sprint 8** | Transfer Antar Dompet | ⏳ |
| **Sprint 9** | Tagihan + Jadwal + Pengingat | ⏳ |
| **Sprint 10** | Tabungan + Target Keuangan | ⏳ |

---

## Definition of Done

- Semua requirement sprint terpenuhi
- Tidak ada error TypeScript
- Build berhasil (`npm run build`)
- CRUD berjalan normal
- Hak akses sesuai role
- Dokumentasi diperbarui (CHANGELOG.md, TASKS.md)
- UI responsif
- Kode mengikuti ARCHITECTURE.md dan CONTRIBUTING.md
